# Eye & Head Gesture Control for YouTube

A Python project that uses eye blinks, gaze direction, and head gestures to 
control YouTube (play/pause, forward, rewind, volume, mute).

## How to Run

1. Install dependencies:

   pip install -r requirements.txt

2. Run the program:

   python3 main.py

Make sure your webcam is working and a YouTube tab is focused.
